โปรเจกต์นี้เป็นส่วนหนึ่งของการฝึกพัฒนา **Frontend Dashboard สำหรับระบบจัดการ Ticket / งานแจ้งปัญหา**  
พัฒนาขึ้นเพื่อเรียนรู้โครงสร้างการเขียนโค้ดแบบที่ใช้จริงในงาน

โปรเจกต์ยังอยู่ในระหว่างการพัฒนา 
โดยเริ่มจากฝั่ง Frontend ก่อน และใช้ข้อมูลจำลอง (Mock Data) แทน Backend

---

## เป้าหมายของโปรเจกต์
- ฝึกออกแบบหน้า Dashboard ให้แสดงข้อมูลภาพรวมได้เข้าใจง่าย
- ฝึกแยกโครงสร้างโค้ดให้เป็นระบบ (Action / Service / Types / Components)
- เตรียมโครงสร้างให้สามารถเชื่อมต่อ Backend จริงได้ในอนาคต
- ใช้เป็นผลงานประกอบการสมัครงานตำแหน่ง Frontend / Junior Developer

---

## ฟีเจอร์ที่ทำแล้ว (Current Features)

- หน้า Dashboard แสดงภาพรวมสถานะของ Ticket
- กราฟวงกลม (Pie Chart) แสดงสัดส่วนของงานแต่ละสถานะ
  - รอดำเนินการ
  - กำลังทำ
  - เสร็จแล้ว
- โครงสร้างรองรับการเลือกช่วงวันที่ (Date Range)
- ใช้ Mock Data เพื่อพัฒนา UI ระหว่างรอ Backend
